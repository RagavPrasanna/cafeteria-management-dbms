# Cafeteria Management System

A comprehensive Database Management System (DBMS) designed to manage, store, and manipulate data for College cafeteria. This project covers database design, schema creation, normalization, and complex querying.

---

## Features
* **Relational Database Design:** Fully normalized schema up to 3NF (Third Normal Form).
* **Data Integrity:** Implementation of primary keys, foreign keys, and cascading deletes.
* **Complex Queries:** Optimized joins, subqueries, and aggregate functions for reporting.
* **Automation:** Stored procedures, triggers, and views for automated workflows.
* **UI/API:** Connected via Node.js  to a front-end interface.
